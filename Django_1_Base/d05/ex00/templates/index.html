Markdown Cheatsheet

Headings

Use # for headings:

# Heading 1
## Heading 2
### Heading 3

Bold and Italics

Bold: **bold text** or __bold text__

Italics: *italic text* or _italic text_

Lists

Unordered List:

* Item 1
* Item 2

Ordered List:

1. First item
2. Second item

Links

Example: [Link Text](URL)

Images

Example: ![Alt Text](Image URL)

Blockquotes

Example: > Blockquote

Code

Inline Code: `code`

Code Block:

```
function example() {
    console.log("Hello World");
}
```

Tables

Example:

| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Row 2    |
